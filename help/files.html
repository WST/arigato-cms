Управление файлами

Данный пункт панели администрирования позволяет управлять файлами сайта через WEB-интерфейс. Доступны три базовые папки: папка с файлами (files), папка с картинками (images) и папка с дополнительными файлами (other). Администратор может загружать файлы со своего компьютера на сервер в любую из трех папок, переименовывать или удалять уже существующие файлы, создавать подпапки, а так же копировать файлы из одной папки в другую.

Каждая из трех папок имеет свое предназначение:

Папка files должна содержать файлы, которые будут доступны пользователям сайта для скачивания. Ядро системы Arigato CMS позволяет автоматически создавать счетчики скачивания для всех файлов в папке files, что позволяет просматривать статистику закачек.

Папка images должна содержать любые рисунки, относящиеся к сайту. В частности, в ней должны размещаться иконки к разделам сайта.

Папка other должна содержать прочие файлы, которые не предоставляются пользователям сайта для скачивания, но и не являются рисунками. К примеру, в ней могут располагаться внешние JavaScript, дополнительные таблицы стилей CSS, вспомогательные PHP-скрипты и прочее.
