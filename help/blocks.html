Управление блоками сайта

Настройка блоков

Этот пункт панели администрирования позволяет производить общие настройки блоков сайта. Можно включать или отключать те или иные блоки: поиск по сайту, подписка на рассылку, письмо администратору сайта. На некоторые блоки можно устанавливать дополнительные права доступа: последняя новость сайта может отображаться все время, в течении срока давности или же вообще отсутствовать; время работы скриптов и статистика сайта могут быть доступны всем, только администратору или отсутствовать. Здесь же задается контент колонтитула сайта и код баннера, который может быть использован для размещения ссылок на другие сайты (кнопки сайтов), счетчиков внешней статистики и прочего.

Блок ссылок

Можно сформировать список ссылок на внешние ресурсы (сайты). Данный список будет отображаться на сайте как меню. Для добавления новой ссылки нужно указать название ресурса и его URL-адрес. Уже добавленным ссылкам можно менять порядок вывода или же удалять их.

Блок голосований

Система Arigato CMS позволяет организовывать на сайте любое число голосований (опросов посетителей). В данном разделе администраторской панели отображается список всех голосований. Результаты любого из них можно просмотреть, щелкнув по названию мышкой. Каждое голосование может быть активно (в этом случае оно будет размещено на сайте и пользователи смогут отвечать) либо закрыто (доступно только администратору, на сайте не отображается). Любое из голосований можно удалить, активировать либо закрыть.

Для добавления нового голосования нужно перейти по ссылке «Добавить голосование». Необходимо указать вопрос голосования и варианты ответа. Продолжительность опроса может быть либо бессрочной (пока не будет закрыто администратором), в этом случае нужно указать в качестве продолжительности ноль, либо существовать определенное количество дней, которое и указывается в качестве значения продолжительности. Голосование с указанной продолжительностью по истечению срока автоматически закрывается.

Чат сайта

На сайте можно организовать так называемый мини-чат. Это блок, в котором пользователи сайта могут посылать сообщения. Эти сообщения будут доступны всем другим пользователям, что позволяет реализовывать общение между посетителями.

Если установить флажок «Блок чата на сайте», то мини-чат будет присутствовать, в противном случае – нет.

Особенностью мини-чата является то, что все сообщения хранятся непродолжительное время. Кроме того, устанавливается лимит на максимальное количество сообщений. Если количество сообщений превышает лимит, то самые старые удаляются автоматически. Время хранения истории задает промежуток (в минутах), в течении которого новые сообщения будут видны. По истечению этого времени сообщения удаляются. Так же администратор может установить максимальную длину сообщения.

Чат устроен таким образом, что его содержимое должно постоянно обновляться, что бы загружать новые сообщения. Частота обновления задается здесь же в секундах. Однако любой пользователь сам сможет выбрать режим работы чата: либо автоматическое обновление, либо обновление вручную. На чат распространяются система бана (запрета доступа) по IP-адресу, автоцензура и запрещенные имена из комментариев к статьям.
