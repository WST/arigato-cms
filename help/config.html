Настройка конфигурации

Данный пункт администраторской панели позволяет настроить некоторые параметры сайта. Администратору предоставляется следующая форма, заполненная текущими значениями:

Название сайта – поле содержит название сайта, которое может быть изменено по желанию администратора.

URL сайта – URL-адрес сайта. Необходимо указывать реальный URL-адрес. В большинстве случаев менять данный параметр не имеет смысла, если только сайт не доступен с различных URL-адресов, тогда можно будет выбрать один из них.

Имя скина – поле содержит выпадающий список с перечнем всех установленных в системе скинов (дизайнов сайта). Администратор в любой момент может выбрать один из доступных скинов, выбрав его из списка.

Формат даты – можно изменить формат даты по своему усмотрению. Дата представляется в формате, принятом в PHP и по умолчанию имеет вид «d.m.Y H:i».

Ширина меню – поле содержит значение, указывающее ширину меню. Имеет смысл изменить этот параметр в том случае, если пункты меню не умещаются по ширине в отведенное для них место.

Сообщений на странице в гостевой – количество сообщений на одну страницу в системе комментариев к статьям (или же гостевой книги). Если сообщений больше, чем указано в данном параметре, то они будут разбиваться на несколько страниц.

Сообщений на страницу в истории – аналогично, но для истории обновлений сайта.

Сообщений на страницу в новостях – то же самое, но для новостных лент.

Обновление статистики – число, отображающее время в минутах обновления статистики. Мо умолчанию принимается 30 минут. Это значит, что пользователь, зашедший на сайт, будет считаться просматривающим сайт в течении 30 минут, если он не загрузит новой странички.

Пометка нового раздела – число дней, в течении которых вновь созданный раздел сайта будет считаться новым. Также этот параметр используется для автоматической рассылки обновлений на сайте, в этом случае рассылка будет проходить не чаще указанного здесь значения.

Антифлуд в подписке на рассылку – параметр указывает количество секунд, в течении которых пользователь не может дважды повторить попытку подписки на рассылку. Это позволяет предотвратить спам чужого почтового ящика через систему подписки на рассылку.

Антифлуд в гостевой – количество секунд, в течении которых пользователь не может повторно отправить сообщение в системе комментариев (или же в гостевой книге).

Антифлуд в письме администратору – аналогичный параметр, отражающий количество секунд, в течении которых пользователь не сможет повторно отправить письмо администратору сайта.

Максимальная длина комментария – параметр задает количество символов, которое не может превышать комментарий пользователя к статьям. Максимально возможное значение – 65535 символов.

Ширина и высота эскиза фотографии – задает ширину и высоту в пикселях автоматически формируемых эскизов фотографий в фотоальбомах сайта.

Mail администратора – адрес электронной почты администратора сайта. Данный E-Mail будет автоматически подставляться как адрес отправителя письма при любых рассылках пользователям сайта.

Поле «От» в письмах – параметр, который будет указывать имя, от кого отправляются все письма, автоматически рассылаемые с сайта пользователям.

Подпись администратора – подпись администратора добавляется в конец всех отправляемых писем.

Рассылать новости сайта – если флажок установлен, то всем пользователям, подписавшимся на рассылку сайта, будут автоматически рассылаться новости.
