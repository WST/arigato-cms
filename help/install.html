Установка системы Arigato CMS

Система автоматизации управления контентом сайта Arigato CMS требует использования сервера с установленным на нем интерпретатором PHP 4 или более поздней версии и сервера базы данных MySQL. Требуется, что бы на сервере было не менее 1 Мб свободного дискового пространства, которое необходимо для размещения скриптов системы и вспомогательных элементов. Для установки системы ее необходимо скопировать на сервер в полном объеме, используя FTP. За установку отвечает скрипт install.php, однако, если система еще не установлена, то он запустится автоматически в случае перехода на сайт.

Установка проходит в один этап и требует от администратора указания всех необходимых параметров:

Сервер MySQL – адрес сервера баз данных MySQL.

Логин MySQL – логин (имя) пользователя базы данных.

Пароль MySQL – пароль указанного пользователя.

База данных – имя базы данных указанного пользователя, которая будет использоваться системой. Указанные выше поля уточняют параметры соединения с базой данных MySQL. Их значения должны предоставляться администратором хостинг-провайдера.

Префикс таблиц – необязательный параметр. Указанный префикс будет добавляться в начало имени всех созданных таблиц, это позволит использовать одну базу данных сразу несколькими системами.

Название сайта – название создаваемого сайта, например: "Сайт школы № 1".

Адрес сайта – URL-адрес сайта, рекомендуется параметр оставить по умолчанию.

Email администратора – адрес электронного ящика администратора сайта.

Пароль администратора – пароль на доступ к функциям администратора для управления сайтом.

Повторите пароль – контроль введенного пароля. Название сайта, адрес сайта, Email администратора, а так же пароль администратора в дальнейшем можно будет изменить через администраторскую панель.

Если во время установки возникает ошибка, связанная с невозможностью создания файла, необходимо через FTP открыть доступ на запись для папки, в которой размещаются скрипты системы. Ошибки с подключением к базе данных устраняются уточнением значений полей сервера, логина, пароля MySQL и имени базы данных.

После установки системы будут созданы два новых файла:

conf.php – файл содержит все параметры подключения к базе данных MySQL. Если потребуется изменить какие-либо из этих параметров, то файл можно отредактировать вручную, используя любой FTP-клиент.

install.lock – файл, наличие которого говорит системе о том, что установка уже завершена. Если в дальнейшем появится необходимость произвести повторную установку, но перед запуском скрипта install.php необходимо удалить данный файл. В противном случае скрипт install.php не предоставляет возможностей установки (что сделано с целью предотвращения его запуска злоумышленником).

После успешной установки системы появится возможность войти в панель администратора, используя указанный пароль. Для доступа к самому сайту необходимо создать хотя бы один информационный раздел, первый созданный раздел будет рассматриваться как главная страница сайта и открываться по умолчанию.
